# Api для генерации кошелька

> **Method**: GET
>
> **Action**: /api/v1/wallet/generate

## Пример

#### Запрос
http://localhost/api/v1/wallet/generate

#### Ответ

Status: 200

```
{
	"id": 1,
	"address": "0x611a44A5cCD53A75d666033f609aF8468e07D80e",
	"publicKey": "sadasd",
	"privateKey": "sadasd",
	"message": "Wallet success generated!"
}
```
